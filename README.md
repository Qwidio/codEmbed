# codEmbed
simple code sharer that can be embeded anywhere
[try it now](https://codembed.rf.gd)

## running in local machine
make sure you'd have apache and mysql installed or for more faster and easier method just install XAMPP 3.3.0
- download the project folder in .zip file
- extract .zip file into htdocs folder inside your installed XAMPP directory
- import the codembed.sql to the database
the project can be opened at localhost/codEmbed (asumming the extracted project folder named 'codembed')
